# PenetrationTest代码介绍

### （1）zombie_scan

僵尸扫描脚本，利用僵尸机对目标服务器进行端口扫描以达到隐蔽自身的目的

- 使用了地址欺骗
- 需要自己寻找僵尸机

### （2）traceroute

模拟Linux下的traceroute命令对目标设备进行路由追踪。原理是使用了TTL的生存周期为0时，最终端设备会返回传递的数据包。

### （3）ARPspoofing

局域网内的ARP欺骗

- 发送双工ARP欺骗包，即欺骗网关，又欺骗目标主机
- 此ARP欺骗攻击会让目标主机的进、出流量都经过攻击主机
- 可以利用ARP欺骗结合iptables进行无路由器权限的流量限制
